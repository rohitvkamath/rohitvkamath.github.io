Architecture

This site is a static website generated using Docusaurus 3.

Tech Stack​

  • Framework: Docusaurus 3.0.1
  • Library: React 18
  • Styling: CSS Modules and custom CSS (src/css/custom.css)
  • Content: Markdown (MD) and MDX

Project Structure​

  • /blog: Contains blog posts.
  • /docs: Technical documentation and guides.
  • /src/pages: Main site pages.
    • index.js: The landing page.
    • work.md: The Work Experience page (Markdown-based page).
  • /src/components: Reusable React components and MDX fragments.
    • header/: Custom header component.
    • profile_summary.mdx: The "About Me" section used on the home page.
  • /static: Static assets like images and favicons.

Key Components​

Home Page (src/pages/index.js)​

The home page uses a combination of a React Header component and the ProfileSummary MDX component to render the landing page content.

Work Experience (src/pages/work.md)​

This is a standalone Markdown page that lists professional experience. It is linked directly from the navbar.

Profile Summary (src/components/profile_summary.mdx)​

A central piece of content that provides the "About Me" overview. It uses MDX to allow embedding React-like styling and components within Markdown.
